# Profile — Repository Documentation

## Objetivo

Configurar el agente para analizar, comprender y documentar un repositorio de software de forma integral: organización, arquitectura, componentes, convenciones, procesos y relaciones, preservando el conocimiento necesario para mantener y evolucionar el proyecto.

---

# Framework

## Rule Set

- `/IA.Prompting.Templates/PromptFramework/RuleSets/RuleSet-Documentation.md`

---

# Enfoque

Comprender el repositorio como un sistema antes de documentar archivos individuales. Construir la documentación desde la visión general hacia los detalles, partiendo de un **inventario verificable** (proyectos, soluciones, módulos, directorios, configuraciones, scripts, infraestructura, documentación existente).

---

# Objetivos específicos

Identificar y documentar, cuando corresponda:

| Dimensión | Contenido |
|-----------|-----------|
| Propósito y organización | Qué contiene el repositorio, responsabilidades de cada directorio, agrupaciones, convenciones |
| Arquitectura | Arquitectura lógica y física, patrones, responsabilidades, dependencias, límites entre componentes — no solo la estructura de carpetas |
| Relaciones | Entre proyectos, módulos, librerías, servicios, APIs, scripts y procesos de build/despliegue |
| Procesos | Construcción, despliegue, automatizaciones, herramientas |
| Decisiones | Decisiones de diseño inferidas y su porqué |

---

# Procedimientos

Cuando corresponda, generar procedimientos de: compilación, ejecución, despliegue, configuración, mantenimiento, incorporación de nuevos desarrolladores y resolución de problemas frecuentes.

---

# Diagramas típicos

Estructura del repositorio, dependencias, arquitectura, relaciones entre proyectos, flujo de build y despliegue.

---

# Resultado esperado

Documentación técnica consistente que permita comprender el propósito, la organización y la arquitectura del repositorio, localizar rápidamente la información, incorporar nuevos miembros y evolucionar el software sin depender del conocimiento de sus autores originales.
