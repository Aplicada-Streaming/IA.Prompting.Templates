# RuleSet — Security-Audit

## Objetivo

Reglas para auditorías de seguridad de sistemas web, APIs o aplicaciones: pruebas autorizadas y no destructivas más un informe técnico conforme a estándares de la industria. Extiende `RuleSet-Default` con indexación, delegación, documentación técnica, doble audiencia, voz narrativa y la regla de pruebas de seguridad.

Las restricciones de autorización y los criterios de calidad del hallazgo y del informe los fija el Profile que aplica este RuleSet (`Web-Security-Audit`).
