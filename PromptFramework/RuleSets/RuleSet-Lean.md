# RuleSet — Lean

## Objetivo

Conjunto mínimo para tareas simples, puntuales o de análisis rápido: aplica solo las reglas imprescindibles para trabajar con método sin pagar el costo de la cadena completa.

Es el tier más liviano del framework: carga menos reglas que `RuleSet-Default`. Elegirlo cuando la tarea no genera documentación técnica ni toca la base de conocimiento.
