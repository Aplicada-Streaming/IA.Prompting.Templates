# RuleSet — Solution-Documentation

## Objetivo

Reglas para documentar soluciones de software completas conforme a un marco documental maestro, con orquestación de subagentes por pieza. Extiende `RuleSet-Documentation` con la doble audiencia (humana + agentes de IA) y el control de deriva.

El énfasis y los criterios de calidad los fija el Profile que aplica este RuleSet (`Solution-Documentation`).
