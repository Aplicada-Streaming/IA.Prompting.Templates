# Rule Set — Audit

## Objetivo

Definir el conjunto de reglas que deberán aplicarse durante una auditoría técnica de infraestructura, sistemas, aplicaciones o servicios.

Una auditoría tiene como finalidad obtener una representación fiel del estado actual del sistema, identificar inconsistencias y generar evidencia verificable sin modificar el entorno auditado.

# Objetivos de la auditoría

La auditoría deberá procurar:

- comprender el sistema;
- construir un inventario;
- identificar componentes;
- detectar relaciones entre componentes;
- verificar configuraciones;
- detectar inconsistencias;
- documentar el estado actual;
- preservar evidencia;
- mantener sincronizada la documentación.

---

# Restricciones adicionales

Durante una auditoría:

- no modificar configuraciones;
- no instalar software;
- no eliminar archivos;
- no reiniciar servicios;
- no alterar datos;
- no ejecutar comandos destructivos;
- utilizar únicamente mecanismos de inspección y lectura.

Toda modificación deberá ser únicamente documental.

---

# Prioridades

La ejecución deberá priorizar el siguiente orden:

1. comprender el entorno;
2. construir el inventario;
3. identificar relaciones;
4. verificar configuraciones;
5. documentar;
6. registrar evidencias;
7. actualizar la memoria operativa;
8. actualizar la indexación;
9. validar la consistencia del resultado.

---

# Resultado esperado

Al finalizar la auditoría deberá existir, según corresponda:

- inventario actualizado;
- documentación técnica;
- observaciones;
- inconsistencias detectadas;
- evidencias;
- documentación sincronizada;
- memoria operativa actualizada;
- índices actualizados.

---

# Criterios de calidad

Una auditoría se considerará finalizada únicamente cuando:

- el estado documentado represente el estado real del sistema;
- toda afirmación relevante tenga evidencia verificable;
- las inconsistencias se encuentren documentadas;
- la memoria operativa refleje el estado actualizado de la documentación;
- no existan referencias obsoletas ni documentación inconsistente.
