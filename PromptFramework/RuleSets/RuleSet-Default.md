# RuleSet — Default

## Objetivo

Definir el conjunto base de reglas aplicado en toda operación del Prompt Framework.

Los RuleSets especializados extienden este conjunto incorporando comportamientos adicionales para sus dominios específicos.

---

## Reglas

Las siguientes reglas se aplican en toda ejecución del framework:

| Regla | Descripción |
|-------|-------------|
| `Rule-All.md` | Comportamiento general y principios transversales |
| `Rule-Tasks.md` | Análisis, planificación y descomposición de tareas |
| `Rule-Execution.md` | Ejecución ordenada y verificable |
| `Rule-Documentation.md` | Generación de documentación técnica de calidad |
| `Rule-Markdown.md` | Formato y estructura de documentos Markdown |
| `Rule-Evidences.md` | Trazabilidad y verificabilidad de la información |
| `Rule-Indexing.md` | Gestión de la base de conocimiento |

## Extensión

Los RuleSets especializados extienden este conjunto para sus dominios:

| RuleSet | Énfasis adicional |
|---------|-------------------|
| `RuleSet-Documentation.md` | Documentación técnica estructurada y mantenible |
| `RuleSet-Development.md` | Análisis de código, arquitectura y calidad técnica |
| `RuleSet-Audit.md` | Solo lectura, evidencia verificable y restricciones de auditoría |
