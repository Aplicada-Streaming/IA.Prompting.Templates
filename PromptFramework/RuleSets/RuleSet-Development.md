# RuleSet — Development

## Objetivo

Reglas para tareas de desarrollo de software: revisión de código, análisis de arquitectura y evaluación técnica. Extiende `RuleSet-Default` con la regla de delegación.

El énfasis y los criterios de calidad los fija el Profile que aplica este RuleSet (`Code-Review`).
